# AI 全栈工程师学习知识库

> 2025-2026 年 AI Agent / FDE / LLM 应用开发学习路径

## 简介

本项目是一套系统化的 AI 工程师学习路线图，专为准备进入 AI 领域的开发者设计。知识库基于真实招聘需求分析（BOSS 直聘、拉勾、猎聘等平台），覆盖从入门到就业的完整学习路径。

## 目标岗位

- **AI Agent 工程师** — 2025 年热门岗位
- **Foundation Developer Engineer (FDE)** — 基础开发工程师
- **LLM 应用开发工程师** — 大模型应用开发
- **AI 算法工程师**
