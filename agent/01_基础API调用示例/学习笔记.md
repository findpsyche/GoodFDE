## 核心概念笔记

### 1. API调用基础
```python
# OpenAI调用模式
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "..."}]
)

# Anthropic调用模式
message = client.messages.create(
    model="claude-3-5-haiku-20241022",
    max_tokens=1024,  # 必需参数
    messages=[{"role": "user", "content": "..."}]
)
#必须的是message model openai : response  anthropic: message
```

### 2. 重要参数
- **temperature**: 0-2，控制随机性
  - 0: 确定性输出，适合代码生成
  - 1: 平衡
  - 2: 高创意性

- **max_tokens**: 限制输出长度
  - OpenAI: 可选
  - Anthropic: 必需

- **system message**: 设定AI角色
  - OpenAI: 在messages中
  - Anthropic: 单独的system参数

### 4. Token和成本
- 1 token ≈ 0.75个英文单词
- 1 token ≈ 1.5-2个中文字符
- 成本 = (输入tokens × 输入价格 + 输出tokens × 输出价格) / 1M

### 5. 模型选择
**OpenAI:**
- gpt-4o: 最强，贵
- gpt-4o-mini: 平衡
- gpt-3.5-turbo: 便宜快速

**Anthropic:**
- claude-opus: 最强
- claude-sonnet: 平衡
- claude-haiku: 最快最便宜
