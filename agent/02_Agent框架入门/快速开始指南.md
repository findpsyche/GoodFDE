## 🛠️ 自定义工具开发

### 工具目录结构
```
tools/
├── __init__.py          # 工具包初始化
├── calculator.py        # 计算器工具
├── weather.py          # 天气工具
└── search.py           # 搜索工具
```

### 创建自定义工具

#### 1. 创建工具文件
```python
# tools/my_tool.py

class MyTool:
    """自定义工具类"""

    def __init__(self):
        """初始化"""
        pass

    def execute(self, input_text: str) -> str:
        """
        执行工具功能

        Args:
            input_text: 输入文本

        Returns:
            处理结果
        """
        try:
            # 你的工具逻辑
            result = f"处理结果: {input_text}"
            return result
        except Exception as e:
            return f"错误: {str(e)}"

    def get_description(self) -> str:
        """获取工具描述"""
        return """我的自定义工具 - 功能描述

使用方法:
  输入格式说明

示例:
  示例1
  示例2
"""

# 便捷函数
def my_tool(input_text: str) -> str:
    tool = MyTool()
    return tool.execute(input_text)
```

#### 2. 在Agent中使用
```python
from langchain.agents import Tool
from tools.my_tool import my_tool

tools = [
    Tool(
        name="MyTool",
        func=my_tool,
        description="清晰的工具描述，说明何时使用此工具"
    )
]
```

## 🐛 常见问题排查

### 问题1: 导入错误
```
ImportError: cannot import name 'ChatOpenAI'
```

**解决方案**：
```bash
pip install --upgrade langchain langchain-openai
```

### 问题2: API密钥错误
```
AuthenticationError: Invalid API key
```

**解决方案**：
1. 检查 `.env` 文件是否存在
2. 确认API密钥正确
3. 确认密钥有足够的额度

### 问题3: Agent陷入循环
```
Agent stopped due to max iterations
```

**解决方案**：
1. 检查工具描述是否清晰
2. 降低任务复杂度
3. 增加 `max_iterations` 参数

### 问题4: 工具选择错误
```
Agent选择了错误的工具
```

**解决方案**：
1. 改进工具描述
2. 提供输入格式示例
3. 减少工具数量

### 问题5: Memory占用过多
```
Token limit exceeded
```

**解决方案**：
1. 使用 `ConversationBufferWindowMemory`
2. 减小窗口大小 `k=3`
3. 使用 `ConversationSummaryMemory`

## 📊 性能优化

### 1. 减少LLM调用
```python
# 使用缓存
from langchain.cache import InMemoryCache
langchain.llm_cache = InMemoryCache()

# 批量处理
results = chain.batch([input1, input2, input3])
```

### 2. 并发处理
```python
import asyncio

# 异步调用
async def process_multiple():
    tasks = [chain.ainvoke(input) for input in inputs]
    results = await asyncio.gather(*tasks)
    return results
```

### 3. 选择合适的模型
```python
# 简单任务用便宜模型
simple_llm = ChatOpenAI(model="gpt-4o-mini")

# 复杂任务用强大模型
complex_llm = ChatOpenAI(model="gpt-4o")
```
