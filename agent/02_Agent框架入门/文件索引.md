## 🛠️ 工具包说明

### tools/calculator.py
**功能**: 数学计算工具
**支持**:
- 基本运算: +, -, *, /, **
- 数学函数: sqrt, sin, cos, tan, log, exp
- 常量: pi, e

**使用**:
```python
from tools.calculator import calculate
result = calculate("sqrt(16) + 2**3")
```

---

### tools/weather.py
**功能**: 天气查询工具
**模式**:
- 模拟数据（默认）
- 真实API（配置OPENWEATHER_API_KEY）

**支持城市**: 北京、上海、深圳、广州、成都、杭州、南京、武汉

**使用**:
```python
from tools.weather import get_weather
result = get_weather("北京")
```

---

### tools/search.py
**功能**: 搜索工具
**支持**:
- DuckDuckGo网络搜索
- Wikipedia知识查询
- 自动选择最佳来源

**使用**:
```python
from tools.search import search
result = search("人工智能")
```
