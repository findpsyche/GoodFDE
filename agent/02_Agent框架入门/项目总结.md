## 💡 核心知识点总结

### 1. Agent vs Workflow

| 维度 | Workflow | Agent |
|------|----------|-------|
| 执行方式 | 固定步骤 | 动态决策 |
| 灵活性 | 低 | 高 |
| 成本 | 可预测 | 变动 |
| 稳定性 | 高 | 中 |
| 适用场景 | 固定流程 | 需要工具调用 |

**选择原则**：
- 能用Workflow就不用Agent
- 需要工具调用才用Agent
- 成本敏感优先Workflow

### 2. Memory类型选择

| Memory类型 | 适用对话长度 | Token消耗 | 推荐场景 |
|-----------|-------------|----------|---------|
| BufferMemory | <10轮 | 高 | 短对话 |
| WindowMemory | 10-50轮 | 中 | 中等对话 |
| SummaryMemory | >50轮 | 低 | 长对话 |
| SummaryBufferMemory | >50轮 | 中 | 平衡方案 |

### 3. 工具设计原则

1. **单一职责** - 每个工具只做一件事
2. **清晰描述** - 详细说明功能和输入格式
3. **错误处理** - 完善的异常处理
4. **类型提示** - 使用类型注解

### 4. 成本控制策略

```python
# 1. 设置最大迭代次数
agent_executor = AgentExecutor(
    agent=agent,
    tools=tools,
    max_iterations=5  # 限制迭代
)

# 2. 使用便宜模型
llm = ChatOpenAI(model="gpt-4o-mini")

# 3. 使用缓存
from langchain.cache import InMemoryCache
langchain.llm_cache = InMemoryCache()

# 4. 优先使用Workflow
# 固定流程不要用Agent
```

## 📈 代码示例总结

### 1. 创建基础Agent
```python
from langchain_openai import ChatOpenAI
from langchain.agents import AgentExecutor, create_react_agent, Tool
from langchain import hub

# 初始化LLM
llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)

# 定义工具
tools = [
    Tool(
        name="Calculator",
        func=calculate,
        description="计算数学表达式"
    )
]

# 创建Agent
prompt = hub.pull("hwchase17/react")
agent = create_react_agent(llm, tools, prompt)

# 创建执行器
agent_executor = AgentExecutor(
    agent=agent,
    tools=tools,
    verbose=True,
    max_iterations=5
)

# 执行
result = agent_executor.invoke({"input": "计算 2+2"})
```

### 2. 创建Workflow
```python
from langchain.chains import LLMChain, SequentialChain
from langchain.prompts import PromptTemplate

# 定义步骤
chain1 = LLMChain(llm=llm, prompt=prompt1, output_key="output1")
chain2 = LLMChain(llm=llm, prompt=prompt2, output_key="output2")

# 组合
workflow = SequentialChain(
    chains=[chain1, chain2],
    input_variables=["input"],
    output_variables=["output1", "output2"]
)

# 执行
result = workflow.invoke({"input": "..."})
```

### 3. 添加Memory
```python
from langchain.memory import ConversationBufferWindowMemory
from langchain.chains import ConversationChain

# 创建Memory
memory = ConversationBufferWindowMemory(k=5)

# 创建对话链
conversation = ConversationChain(
    llm=llm,
    memory=memory,
    verbose=True
)

# 对话
response = conversation.predict(input="你好")
```

### 4. 自定义工具
```python
from langchain.agents import Tool

def my_function(input_text: str) -> str:
    """工具函数"""
    return f"处理结果: {input_text}"

tool = Tool(
    name="MyTool",
    func=my_function,
    description="清晰的工具描述，说明何时使用"
)
```

## 🐛 常见问题和解决方案

### 1. Agent陷入循环
**原因**：
- 工具描述不清晰
- 任务过于复杂
- 工具返回信息不足

**解决方案**：
- 优化工具描述
- 简化任务
- 设置max_iterations
- 改进工具返回信息

### 2. 工具选择错误
**原因**：
- 工具描述模糊
- 工具功能重叠
- Prompt不够清晰

**解决方案**：
- 提供详细描述和示例
- 确保工具职责单一
- 优化System Prompt

### 3. Memory占用过多
**原因**：
- 使用BufferMemory
- 对话轮次太多
- 没有清理历史

**解决方案**：
- 切换到WindowMemory
- 定期清理Memory
- 使用SummaryMemory

### 4. 成本过高
**原因**：
- 过多的Agent调用
- 使用昂贵模型
- 没有缓存机制

**解决方案**：
- 优先使用Workflow
- 选择便宜模型
- 启用缓存
- 设置max_iterations
