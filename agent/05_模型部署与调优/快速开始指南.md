```python
import requests
import json

def test_ollama():
    """测试 Ollama API"""
    url = "http://localhost:11434/api/generate"

    data = {
        "model": "llama3:8b",
        "prompt": "What is the capital of France?",
        "stream": False

        print(f"✅ 响应成功！")

    else:
        print(f"❌ 请求失败: {response.status_code}")

if __name__ == "__main__":
    test_ollama()
```

## 📊 第三步：理解量化（5分钟）

### 什么是量化？

量化是将模型权重从高精度（FP16）转换为低精度（INT4/INT8）的过程：

```
原始模型 (FP16)  →  量化模型 (Q4)
  13GB              4.7GB
  慢                快
  高质量            稍低质量
```

### 量化级别对比

| 量化级别 | 大小 | 速度 | 质量 | 推荐场景 |
|---------|------|------|------|----------|
| Q2_K    | 最小 | 最快 | 较差 | 测试用 |
| Q4_K_M  | 小   | 快   | 好   | **推荐** |
| Q5_K_M  | 中   | 中   | 很好 | 质量优先 |
| Q6_K    | 大   | 慢   | 优秀 | 高质量需求 |
| Q8_0    | 很大 | 很慢 | 极好 | 接近原始 |

### 测试不同量化

```bash
# 下载不同量化版本
ollama pull llama3:8b-q4_K_M  # 默认，已下载
ollama pull llama3:8b-q5_K_M  # 更高质量
ollama pull llama3:8b-q6_K    # 最高质量

# 对比大小
ollama list
```

创建对比脚本 `compare_quantization.py`：

```python
import requests
import time

def test_model(model_name, prompt):
    """测试单个模型"""
    url = "http://localhost:11434/api/generate"

    data = {
        "model": model_name,
        "prompt": prompt,
        "stream": False
    }

    start = time.time()
    response = requests.post(url, json=data)
    elapsed = time.time() - start

    if response.status_code == 200:
        result = response.json()
        return {
            "response": result['response'],
            "time": elapsed,
            "tokens": len(result['response'].split())
        }
    return None

def compare_quantizations():
    """对比不同量化版本"""
    models = [
        "llama3:8b-q4_K_M",
        "llama3:8b-q5_K_M",
        "llama3:8b-q6_K"
    ]

    prompt = "Explain quantum computing in one sentence."

    print("🔬 量化对比测试\n")
    print(f"📝 测试问题: {prompt}\n")

    for model in models:
        print(f"🧪 测试 {model}...")
        result = test_model(model, prompt)

        if result:
            print(f"  ⏱️  耗时: {result['time']:.2f}s")
            print(f"  📊 速度: {result['tokens']/result['time']:.1f} tokens/s")
            print(f"  📝 回答: {result['response'][:100]}...")
            print()

if __name__ == "__main__":
    compare_quantizations()
```

运行对比：

```bash
python compare_quantization.py
```

## 🔍 第四步：性能测试（5分钟）

创建性能测试脚本 `benchmark.py`：

```python
import requests
import time
import statistics

def benchmark_model(model_name, num_runs=5):
    """性能基准测试"""
    url = "http://localhost:11434/api/generate"

    prompts = [
        "What is AI?",
        "Explain machine learning.",
        "What is deep learning?",
        "Define neural networks.",
        "What is NLP?"
    ]

    times = []
    speeds = []

    print(f"🏃 测试 {model_name}...")

    for i, prompt in enumerate(prompts[:num_runs], 1):
        data = {
            "model": model_name,
            "prompt": prompt,
            "stream": False
        }

        start = time.time()
        response = requests.post(url, json=data)
        elapsed = time.time() - start

        if response.status_code == 200:
            result = response.json()
            tokens = len(result['response'].split())
            speed = tokens / elapsed

            times.append(elapsed)
            speeds.append(speed)

            print(f"  Run {i}: {elapsed:.2f}s, {speed:.1f} tokens/s")

    print(f"\n📊 统计结果:")
    print(f"  平均耗时: {statistics.mean(times):.2f}s")
    print(f"  平均速度: {statistics.mean(speeds):.1f} tokens/s")
    print(f"  最快速度: {max(speeds):.1f} tokens/s")
    print(f"  最慢速度: {min(speeds):.1f} tokens/s")

if __name__ == "__main__":
    benchmark_model("llama3:8b")
```

运行测试：

```bash
python benchmark.py
```

## 🎯 第五步：构建简单应用（5分钟）

创建一个简单的命令行聊天机器人 `chatbot.py`：

```python
import requests
import json

def chat(model="llama3:8b"):
    """简单的命令行聊天机器人"""
    url = "http://localhost:11434/api/generate"

    print("🤖 本地聊天机器人启动！")
    print("💡 输入 'quit' 退出\n")

    while True:
        # 获取用户输入
        user_input = input("你: ").strip()

        if user_input.lower() in ['quit', 'exit', 'bye']:
            print("👋 再见！")
            break

        if not user_input:
            continue

        # 调用模型
        data = {
            "model": model,
            "prompt": user_input,
            "stream": False
        }

        print("🤔 思考中...", end="", flush=True)
        response = requests.post(url, json=data)
        print("\r", end="")  # 清除"思考中"

        if response.status_code == 200:
            result = response.json()
            print(f"🤖: {result['response']}\n")
        else:
            print(f"❌ 错误: {response.status_code}\n")

if __name__ == "__main__":
    chat()
```

运行聊天机器人：

```bash
python chatbot.py
```

## 🐛 常见问题

### Q1: Ollama 服务无法启动
```bash
# 检查服务状态
ollama serve

# 如果端口被占用，更改端口
OLLAMA_HOST=0.0.0.0:11435 ollama serve
```

### Q2: 模型下载失败
```bash
# 检查网络连接
ping ollama.com

# 使用代理（如果需要）
export HTTP_PROXY=http://proxy:port
export HTTPS_PROXY=http://proxy:port
ollama pull llama3:8b
```

### Q3: 内存不足
```bash
# 使用更小的模型
ollama pull llama3:8b-q2_K  # 最小版本

# 或使用更小的模型
ollama pull phi3:mini  # 3.8B 参数
```

### Q4: 推理速度慢
- 确保使用 GPU（如果有）
- 使用更激进的量化（Q4 → Q3）
- 减少 prompt 长度
- 关闭其他占用资源的程序
