## 🚀 快速安装

### 步骤 1：检查 Python 版本

```bash

# 应该显示 Python 3.10.x 或更高
```

如果版本过低，请从 [python.org](https://www.python.org/downloads/) 下载安装。

```bash
# 创建虚拟环境
python -m venv dl_env

# 激活虚拟环境
# Windows
dl_env\Scripts\activate

source dl_env/bin/activate
```

**有 NVIDIA GPU（推荐）**

### 3. 完整验证脚本

创建文件 `verify_installation.py`：

```python
import sys

def check_package(package_name, import_name=None):
    """检查包是否安装"""
    if import_name is None:
        import_name = package_name

    try:
        module = __import__(import_name)
        version = getattr(module, '__version__', 'unknown')
        print(f"✅ {package_name}: {version}")
        return True
    except ImportError:
        print(f"❌ {package_name}: 未安装")
        return False

print("=" * 50)
print("深度学习环境验证")
print("=" * 50)

print(f"\nPython 版本: {sys.version.split()[0]}")

print("\n核心库:")
packages = [
    ("torch", "torch"),
    ("torchvision", "torchvision"),
    ("transformers", "transformers"),
    ("datasets", "datasets"),
    ("tokenizers", "tokenizers"),
]

all_installed = True
for pkg, imp in packages:
    if not check_package(pkg, imp):
        all_installed = False

print("\n辅助工具:")
tools = [
    ("tensorboard", "tensorboard"),
    ("jupyter", "jupyter"),
    ("matplotlib", "matplotlib"),
    ("pandas", "pandas"),
    ("scikit-learn", "sklearn"),
]

for pkg, imp in tools:
    check_package(pkg, imp)

# GPU 检查
print("\nGPU 状态:")
import torch
if torch.cuda.is_available():
    print(f"✅ CUDA 可用: {torch.version.cuda}")
    print(f"   GPU 数量: {torch.cuda.device_count()}")
    print(f"   GPU 名称: {torch.cuda.get_device_name(0)}")
else:
    print("⚠️  未检测到 GPU，将使用 CPU")

print("\n" + "=" * 50)
if all_installed:
    print("✅ 所有核心库安装成功！")
else:
    print("❌ 部分库未安装，请检查上述输出")
print("=" * 50)
```

运行：
```bash
python verify_installation.py
```

## 🎯 运行第一个示例

### 示例 1：PyTorch 基础

创建文件 `01_pytorch_basics.py`：

```python
import torch
import torch.nn as nn

# 1. 创建 Tensor
print("1. Tensor 操作")
x = torch.randn(3, 4)
print(f"随机 Tensor:\n{x}\n")

# 2. 自动微分
print("2. 自动微分")
x = torch.tensor([2.0], requires_grad=True)
y = x ** 2 + 3 * x + 1
y.backward()
print(f"y = x^2 + 3x + 1")
print(f"x = {x.item()}, dy/dx = {x.grad.item()}")
print(f"理论值: 2*{x.item()} + 3 = {2*x.item() + 3}\n")

# 3. 简单神经网络
print("3. 简单神经网络")
model = nn.Sequential(
    nn.Linear(10, 20),
    nn.ReLU(),
    nn.Linear(20, 1)
)

x = torch.randn(5, 10)  # batch_size=5, input_dim=10
y = model(x)
print(f"输入形状: {x.shape}")
print(f"输出形状: {y.shape}")
print(f"参数数量: {sum(p.numel() for p in model.parameters())}")

print("\n✅ PyTorch 基础示例完成！")
```

### 示例 2：Transformer 推理

创建文件 `02_transformer_inference.py`：

```python
from transformers import pipeline

print("加载模型...")
# 文本分类
classifier = pipeline("sentiment-analysis")

texts = [
    "I love this movie!",
    "This is terrible.",
    "It's okay, not great.",
]

print("\n情感分析:")
for text in texts:
    result = classifier(text)[0]
    label = result['label']
    score = result['score']
    print(f"  '{text}'")
    print(f"  → {label} (置信度: {score:.4f})\n")

# 文本生成
print("加载生成模型...")
generator = pipeline("text-generation", model="gpt2")

prompt = "Once upon a time"
print(f"\n文本生成 (提示: '{prompt}'):")
result = generator(prompt, max_length=50, num_return_sequences=1)[0]
print(f"  {result['generated_text']}")

print("\n✅ Transformer 推理示例完成！")
```

### 示例 3：训练简单模型

创建文件 `03_simple_training.py`：

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 1. 准备数据（简单的线性回归）
torch.manual_seed(42)
X = torch.randn(100, 1)
y = 3 * X + 2 + torch.randn(100, 1) * 0.1

# 2. 定义模型
model = nn.Linear(1, 1)

# 3. 定义损失函数和优化器
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 4. 训练
print("开始训练...")
for epoch in range(100):
    # 前向传播
    y_pred = model(X)
    loss = criterion(y_pred, y)

    # 反向传播
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

    if (epoch + 1) % 20 == 0:
        print(f"Epoch {epoch+1}/100, Loss: {loss.item():.4f}")

# 5. 查看结果
print(f"\n训练完成！")
print(f"真实参数: w=3, b=2")
print(f"学到的参数: w={model.weight.item():.4f}, b={model.bias.item():.4f}")

print("\n✅ 训练示例完成！")
```

运行：
```bash
python 03_simple_training.py
```

## ❓ 常见问题

### 1. CUDA 版本不匹配

**问题**：
```
RuntimeError: CUDA error: no kernel image is available for execution on the device
```

**解决**：
- 检查 GPU 驱动版本：`nvidia-smi`
- 安装匹配的 PyTorch 版本
- 访问 https://pytorch.org/get-started/locally/ 选择正确版本

### 2. 显存不足（OOM）

**问题**：
```
RuntimeError: CUDA out of memory
```

**解决**：
- 减小 batch size
- 使用梯度累积
- 启用混合精度训练
- 使用更小的模型

### 3. 模型下载失败

**问题**：
```
OSError: Can't load model from 'bert-base-uncased'
```

**解决**：
- 设置镜像源：
```python
import os
os.environ['HF_ENDPOINT'] = 'https://hf-mirror.com'
```
- 或手动下载模型到本地

### 4. 导入错误

**问题**：
```
ImportError: cannot import name 'xxx'
```

**解决**：
- 检查包版本：`pip list | grep torch`
- 升级到最新版本：`pip install --upgrade torch transformers`
- 重新安装：`pip uninstall torch && pip install torch`

### 5. Windows 上 PyTorch 安装慢

**解决**：
- 使用清华镜像源：
```bash
pip install torch -i https://pypi.tuna.tsinghua.edu.cn/simple
```
- 或直接下载 wheel 文件安装

### 6. Jupyter Notebook 找不到虚拟环境

**解决**：
```bash
# 在虚拟环境中安装 ipykernel
pip install ipykernel

# 添加到 Jupyter
python -m ipykernel install --user --name=dl_env --display-name="Deep Learning"
```
