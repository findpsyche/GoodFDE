---

## 总览

| 阶段 | 天数 | 主题 | 重点 |
|------|------|------|------|
| Phase 1 | Day 1-5 | 基础数据结构 | 数组、链表、哈希表、字符串、双指针 |
| Phase 2 | Day 6-10 | 核心算法 | 栈与队列、二叉树、回溯算法 |
| Phase 3 | Day 11-16 | 高级算法 | 贪心、动态规划、单调栈、图论 |
| Phase 4 | Day 17-20 | 模拟冲刺 | 综合模拟、字节高频题、限时训练 |

### 每日时间安排

| 时段 | 时间 | 内容 |
|------|------|------|
| 上午 | 9:00-12:00 | 知识点学习 + 基础题 |
| 下午 | 14:00-18:00 | 核心题目练习 + MIKE框架训练 |
| 晚上 | 19:30-22:30 | 复习 + 补充题 + 自检 |

### MIKE框架速记

```
M - Minimally sketch naive solution（暴力解，2分钟）
I - Identify upper/lower bounds（边界分析，确定可行复杂度区间）
K - Keywords/triggers（关键词触发：已排序→二分、第K个→堆、连通→并查集...）
E - Employ boosters（优化：Memo/DP、剪枝、更好数据结构、二分答案）
```

### B. MIKE关键词触发器速查表

|------------|---------|

| 子串/子数组+条件 | 滑动窗口 |
| 子序列 | DP |
| 所有组合/排列/子集 | 回溯 |
| 最优解+重叠子问题 | DP |
| 区间问题 | 排序+贪心 |
| 下一个更大/更小 | 单调栈 |
| 括号匹配/嵌套 | 栈 |
| 前缀和 | 哈希表存前缀和 |
| 原地操作数组 | 双指针 / 原地哈希 |
| O(1)查找+O(1)删除 | 哈希表+链表 |
